# Open-Set-Vehicle-Recognition-Using-MSTAR-SAR-Images-with-MBCONV-and-OpenMax
This project presents a deep learning-based system for vehicle recognition using Synthetic Aperture Radar (SAR) images. The system uses a lightweight MBCONV architecture for efficient feature extraction and integrates the OpenMax algorithm for open-set recognition. Unlike traditional models, it can detect unknown inputs and classify them as UNKNOWN.
This project focuses on solving the problem of SAR vehicle recognition using deep learning techniques. Synthetic Aperture Radar (SAR) images are widely used in defense and surveillance systems due to their ability to capture images in all weather conditions. However, SAR images are noisy and difficult to interpret.
To address this, the project implements a lightweight deep learning model based on MBCONV (Mobile Inverted Bottleneck Convolution), which provides efficient feature extraction with fewer parameters. In addition, the OpenMax algorithm is integrated to enable open-set recognition, allowing the system to detect unknown inputs instead of misclassifying them.
The system is trained on the MSTAR dataset containing multiple vehicle classes such as 2S1, BRDM_2, BTR_60, D7, T62, ZIL131, and ZSU_23_4. It achieves high accuracy and strong unknown detection capability.
A user-friendly interface is developed using Gradio, allowing users to upload SAR images and obtain real-time predictions along with confidence scores and probability distributions. The system also includes performance evaluation tools such as confusion matrix and accuracy graphs.
This project demonstrates the effectiveness of combining lightweight architectures with open-set recognition techniques for real-world applications in defense and surveillance.
